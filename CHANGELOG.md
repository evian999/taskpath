# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

### Added

- 画布工具栏：「适应任务」一键将视口缩放到当前可见任务；工具栏可左右折叠（文件夹筛选右侧为折叠按钮）；工具栏整体包在带边框的容器内。
- 画布：建组使用图标；横向/纵向等距排列合并为「排列…」下拉。
- `CHANGELOG.md` 变更记录。

### Changed

- 画布任务可见性：未完成任务始终显示；已完成且无出边（无「下一步」连线）不显示；已完成但若沿 **source→target** 能到达至少一个未完成任务则显示，否则整条已完成链不显示。
- 画布：不渲染当前筛选下**没有任何可见任务/任务组**的文件夹竖条（空文件夹不占画布）。
- 文件夹合并布局：包络与任务组框仅统计**画布可见**任务；仅未完成参与扩张的规则改为按可见集统一处理；「文件夹内仅有已完成（画布均不可见）」时使用默认文件夹尺寸；允许将文件夹缩得小于原内容区（不强制向右下扩张）；空文件夹最小约 160×160。
- 画布「全部文件夹」视图下自动消除文件夹 lane 重叠（含加载/切换导航时）。
- 列表模式：「已完成」区块默认折叠。
- 画布连线：选中高亮时不显示线上文字标签。
- 文件夹 lane：修复 `pointerEvents` 导致无法缩放；缩放控件增加 `nopan`；自动扩展留白加大。

### Removed

- 画布工具栏「断开连线」按钮（仍可用 Delete / Backspace 删除选中连线）。

### Fixed

- 任务组外框随成员（可见）自适应；多种 store 路径传入画布可见任务集以正确合并 `folderRects`。
